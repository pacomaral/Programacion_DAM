- Nueva ventana para el modo de juego difícil (v4)

- Se adapta el método sePuedeJugar() para el nuevo modo de juego difícil, aunque sigue siendo posible por la infinidad de posibilidades que no se pueda llegar al número objetivo.
Por esto, se podrá presionar comprobar para pasar a la siguiente jugada.

- Añadido un control de excepciones en el método crearApodo() de la clase jugador, y en caso de que ocurra un error al crear el apodo, se asignará el nombre 
introducido como apodo.

- Añadidas las Inner class fuera del constructor para que en este solo se incluyan los elementos gráficos y el código necesario.

- Se ha añadido un temporizador mediante la clase Timer (en el modo difícil), para que al darle empezar tengamos 1 minuto para jugar, y al acabar, nos mostrará la puntuación mediante
un JOptionPane.
